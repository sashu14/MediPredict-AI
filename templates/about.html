About MediPredict AI

Project Overview

MediPredict AI is a state-of-the-art disease prediction system that utilizes multiple machine learning algorithms to provide accurate health insights based on user-reported symptoms.

Techniques used: Random Forest (Primary), Naive Bayes, and Support Vector Machines (SVM). The system uses model voting to increase reliability.

Tech Stack

Python Flask Scikit-Learn Pandas Bootstrap 5 Chart.js

Dataset Credit

This project uses the "Disease Prediction using Machine Learning" dataset from Kaggle, which includes 4920 records of symptoms for 41 unique diseases.
